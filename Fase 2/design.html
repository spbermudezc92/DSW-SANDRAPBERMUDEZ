Propósito del sitio

En este sitio web se encuentra la definición de los términos Flex Box, lista y enlace los cuales son utilizados para el desarrollo del lenguaje HTML5, dichos términos son conceptos básicos para el desarrollo de un sitio web

FLEX BOX

Definición

Flexbox es un módulo de diseño de CSS3 el cual permite alterar el ancho, alto y posicionamiento de elementos de la mejor manera con el espacio del que disponemos. Dicha etiqueta fue creada para mejorar la forma en la que se hace diseño responsive, evitando así el uso de float, logrando con ello que se escribiera menos código y facilitando el posicionamiento de elementos.

Estrutura

La estructura de Flexbox se compone de contenedores padre e hijos (Contenedor-Flex y Elementos-Flex respectivamente)

- El Contenedor-Flex:

Es nuestro contenedor padre, y es él quién va a contener a todos los elementos hijos a los cuáles queremos posicionar.

- El Elemento-Flex :

Es nuestro elemento hijo, el cual utilizará todo el espacio disponible para ubicarse de acuerdo a las propiedades a las cuales este sometido junto a los demas Elementos-Flex contenidos.

Listas

Definicion

Las listas en HTML permite crear un conjunto de elementos dentro de una página web, dichos elementos generalmente van con una viñeta o número, hay tres tipos diferentes de listas los cuales son:

Listas ordenadas

Estas listas son aquellas que muestran los elementos en orden, es decir cada elemento ira con un número o letra que establecerá su orden.
Dicha lista está representada por el elemento OL.

Lista desordenada

Estas listas son aquellas que se utilizan para mostrar los elementos sin ningún tipo de orden simplemente se le genera una viñeta
Dicha lista está representada por el elemento UL.

Lista de definiciones

Esta lista son aquellas que muestran la estructura valor y definición como se evidencia en esta lista de términos
Dicha lista está representada por el elemento DL.

Enlaces

Definición

Un enlace o Link es un texto o imagen en el cual haciendo clic encima de el nos dirigirá hacia otra página ubicada en la misma página web o en otra de internet, este utiliza la etiqueta a y el atributo href.
hay diferentes tipos de enlaces los cuales son:

Enlace de texto

Es un enlace que se encuentra asociado a un texto

Enlace de imagen

Es un enlace que se encuentra asociado a una imagen

Enlace local

Un enlace local es un vínculo que apunta a una página que se encuentra en el mismo sitio web (sitio local).

Enlace externo

Un enlace externo es un vínculo que apunta a otro sitio web de Internet

Enlace a una dirección

Es un enlace especial que contiene una dirección de correo.

Enlace dentro de la propia página

Es un objeto (texto, imagen…) sobre el que se hace clic y el navegador se deslaza por el documento para mostrarnos la información de otro lugar de ese mismo documento.
